Eredmény
Hogyan bővítsd?
  1. Nyisd meg a fájlt és keresd a QUESTION_BANK tömböt.
  2. Adj hozzá új objektumot: 
    {
  id: "uj-001",
  type: "mcq", // "mcq" (egyválaszos/többválaszos), "fill" (kitöltős)
  multi: false, // true = több helyes
  topic: "Jogi alapok",
  text: "Példa kérdés?",
  options: ["A", "B", "C", "D"],
  correct: [1] // helyes index(ek) 0-alapúan; fill esetén correctText: [...]
}
  3. Többválaszos kérdéshez állítsd multi: true-ra, és több indexet adj a correct-hez.
  4. Kitöltős kérdéshez használj type: "fill"-t és add meg a correctText mezőt (több szinonima is lehet).
